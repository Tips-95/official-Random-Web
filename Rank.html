Official Random Web - Rank System

👑 Owner

Current Owner: Tips-95

The Owner has full control over the website.

  • Edit anything on the site.
  • Add or remove Admins.
  • Add or remove Moderators.
  • Delete any tip.
  • Manage all sections.

🛡️ Admin

Admins help manage the website.

  • Delete tips made by Public users.
  • Help keep the site clean.
  • Cannot create new Admins.
  • Cannot create new Moderators.

Only the Owner can add Admins.

🔨 Moderator

Moderators help expand the site.

  • Add tips to all tip sections.
  • Create content for staff sections.
  • Cannot add Admins.
  • Cannot remove users.

Only the Owner can add Moderators.

🌎 Public

All regular users automatically have this rank.

  • Can submit tips only in the "Public Tips" section.
  • Cannot post in staff sections.
  • Cannot delete tips.
  • Cannot manage ranks.

📋 Tip Section Rules

  • Public Tips → Public, Mod, Admin, Owner
  • All Other Sections → Mod, Admin, Owner only
